## What it does

Black Ops III Code Tester is a web-based letter-to-number converter built for Call of Duty: Black Ops III Data Vault style code verification. You can paste in a word or a full phrase, and the tool will map each letter to its numeric value while showing both the per-word breakdown and the overall sum in a compact view.

Since it is delivered as a plain HTML tool in the browser, there is nothing to install. It is aimed at fast lookups and repeated checks during game puzzle solving, while still remaining comfortable to use on both desktop and mobile screens.
